# Gradke Cruise Companion

Production-ready Progressive Web App.

## Upload to GitHub

Upload the CONTENTS of this folder to the root of your repository.

Files that should be visible at the root:
- index.html
- manifest.webmanifest
- service-worker.js
- README.md
- DEPLOYMENT_CHECKLIST.txt
- icons/

Do not upload only the ZIP file.

## Cloudflare Pages

Framework preset: None  
Build command: leave blank  
Output directory: leave blank or `/`

## Install

Open the Cloudflare URL on your phone.  
iPhone: Safari > Share > Add to Home Screen.  
Android: Chrome > Install app / Add to Home screen.
